“Tidak, kakek, kamu salah paham! Saya benar-benar tidak bisa mengendalikan intensitas api ini.” Surga Pencuri Muda masih melanjutkan aksinya.

“Hehehe. Anda berani menipu saya, Anda benar-benar meremehkan saya. Baiklah, aku tidak akan mengambil nyawamu. Biarkan api ini melahap Anda juga. Anda pikir penyempurnaan Gu sangat sederhana dan aman? Anda mencari malapetaka Anda sendiri dengan sengaja gagal menyempurnakan Gu! Mati, mati, ini hasil kegigihanmu, ini ulahmu sendiri.” Sha Xiao mencibir.

Sebelum dia menyelesaikan kata-katanya, api itu segera membubung dan menutupi Surga Pencuri muda.

“Kenapa ini terjadi?” Surga Pencuri Muda ingin menyerang, tetapi api itu memancarkan kekuatan menarik yang sangat besar yang membuatnya tetap di tempat.

Melihat dia akan dikubur dalam api, Fang Yuan menemukan dia bisa mengendalikan tubuh muda Pencuri Surga sekali lagi.

“Mengapa saya merasa seperti sedang membersihkan kekacauan muda Pencuri Surga?” Fang Yuan merasa frustrasi.

Tapi dia tidak panik.

Fang Yuan telah menemukan resep Gu yang diberikan Sha Xiao kepada Surga Pencuri muda.

Ini adalah resep Gu peringkat satu, sementara Fang Yuan adalah quasi-grandmaster jalur penyempurnaan, jadi prinsip-prinsip resep Gu jelas baginya.

Meskipun Surga Pencuri muda telah melakukan kesalahan, dalam pandangan Fang Yuan, itu tidak dapat diperbaiki.

Pasir Gu.

Fang Yuan mengendalikan tubuh muda Pencuri Surga untuk melemparkan Gu pasir ke posisi tertentu dalam api.

Dia tidak mengaktifkan cacing Gu ini, malah melemparkannya sebagai bahan Gu.

Gu yang halus segera menarik api dan segera dibakar menjadi tumpukan bubuk.

Kekuatan tarik api menurun tajam, Fang Yuan mengambil kesempatan ini untuk melompat keluar.

Dia berguling-guling di tanah beberapa kali untuk memadamkan api yang tersisa pada dirinya.

Luka-lukanya parah.

Wajah, leher, lengan, dan area lainnya dari Surga Pencuri Muda dibakar hingga tingkat yang menyedihkan.

Tapi Fang Yuan mengabaikan luka-luka itu, dan memusatkan seluruh perhatiannya pada api itu.

Surga Pencuri Muda masih belum keluar dari bahaya, karena proses penyempurnaan Gu belum benar-benar berhenti.

Jika api ini terus menyala, pasti akan menelan Surga Pencuri muda lagi.

Jadi Fang Yuan mengaktifkan cerobong asap Gu, asap keluar dan menutupi api seukuran baskom.

Lapisan asap menyelimuti api ini.

Saat bahaya berkurang secara bertahap, Fang Yuan merasa hubungannya dengan api melemah dengan cepat.

“Seperti ini, asapnya bisa memadamkan api dan menghentikan penyempurnaan Gu ini. Meskipun akan ada beberapa reaksi, kehidupan tubuh ini akan dipertahankan. ”

Dalam keadaan ini, Fang Yuan telah melakukan semua yang dia bisa.

Lagi pula, Surga Pencuri muda terlalu sembrono, sebenarnya mencoba untuk dengan sengaja menggagalkan penyempurnaan Gu.

Penyempurnaan Gu ini malah menurunkan kekuatannya karena dia sekarang kehilangan Gu pasir yang berharga.

Namun, perubahan terjadi tepat pada saat ini.

Asap itu tiba-tiba terserap ke dalam api.

“Apa yang sedang terjadi?” Bahkan Fang Yuan tidak mengharapkan ini.

Tapi tak lama, ketika dia melihat hantu bayi itu di dalam api, dia menyadari: “Ada masalah dengan bayi ini, konstitusi mereka mungkin tidak biasa, itu cukup aneh!”

Bayi itu telah ditambahkan ke api sebagai bahan Gu.

Fang Yuan telah mengamati dari samping dan tidak memeriksa bayi itu dengan hati-hati, jadi dia tidak tahu apakah ada yang aneh dengannya.

Anomali ini telah melampaui harapan Fang Yuan.

Api melonjak sekali lagi, hantu bayi itu melompat-lompat di dalam api dan menerkam ke arah Surga Pencuri muda.

“Sial, ini berbahaya.” Fang Yuan menghela nafas, jika Surga Pencuri muda meninggal, eksplorasi alam mimpinya juga akan gagal selanjutnya.

Melihat api yang akan menerkam tubuh muda Pencuri Surga, Fang Yuan tiba-tiba mendapat inspirasi.

“Mengerti!”

Dia pertama kali melemparkan cerobong asap Gu.

Asap cerobong asap Gu segera menarik hantu bayi ke dalam api, itu membuat suara mengoceh saat menyerang asap cerobong asap Gu dan mulai memakannya.

Mengambil kesempatan ini, Fang Yuan dengan cepat mundur beberapa langkah, memisahkan diri dari api.

Tapi segera, bayi itu selesai makan cerobong asap Gu dan menyerbu ke arah Fang Yuan lagi.

Fang Yuan tertawa: “Ayo, ada juga air jernih Gu!”

Air jernih Gu dibuang, dan juga dimakan oleh bayinya.

Api tiba-tiba meredup, air jernih Gu berubah menjadi gumpalan air hijau zamrud yang tidak bisa diuapkan oleh api.

Dan api merah berubah menjadi oranye-kuning, diselimuti cahaya. Itu tidak lagi berapi-api seperti sebelumnya, dan sebaliknya sangat tersembunyi.

Air hijau zamrud dan api jingga-kuning bersirkulasi di udara.

Fang Yuan menggerakkan tubuh muda Pencuri Surga untuk maju bukannya mundur, dan pergi di depan air dan api yang aneh ini.

“Sandpit berubah menjadi bubuk, mengkonsolidasikan tubuh. Asap cerobong asap menyelimuti dan menahan api. Air jernih menggulingkan dunia. Menggunakan darah untuk mengintegrasikan yin dan yang. Hebat!”

Fang Yuan menggigit lidahnya dan memuntahkan darahnya ke dalam air dan api yang aneh.

“Eeeek.” Setelah air dan api aneh disentuh oleh darah, tiba-tiba berubah menjadi hantu bayi biru yang berteriak dan menerkam Fang Yuan.

Fang Yuan tidak menghindar, membiarkannya menyerang.

Hantu bayi biru langsung memasuki lubang muda Pencuri Surga, berubah menjadi cacing Gu peringkat dua.

“Apa ini?” Surga Pencuri Muda kembali ke akal sehatnya dan bingung.

“Apa, lagi?!” Fang Yuan terdiam, saat Surga Pencuri muda aman, dia terlempar ke samping dan berubah menjadi pengamat.

“Kamu benar-benar menyempurnakan cacing Gu yang benar-benar baru di bawah serangkaian kebetulan ?!” Sha Xiao juga berteriak kaget.

“Tidak itu salah.” Tak lama, dia menyadari, “Brat, kamu tampaknya memiliki sifat tersembunyi yang hanya mengambil kendali ketika kamu menghadapi bahaya fatal, dan kemudian membuatmu secara intuitif membuat pilihan terbaik. Seperti ini di kompetisi, sepertinya sama dalam penyempurnaan Gu juga. ”

“Kamu naif dan keras kepala, juga bodoh, tetapi sifat tersembunyimu ini cukup pragmatis dan sangat licik, memiliki bakat yang tidak dapat kamu jangkau! Saya cukup mengagumi alam ini, sangat mengesankan!!”

Sha Xiao mulai tertawa.

“Saya menyempurnakan … Gu baru?” Surga Pencuri Muda masih belum sepenuhnya bereaksi terhadap situasi ini dan bingung.

“Itu benar, cobalah. Gu baru ini adalah Gu peringkat dua, tampaknya merupakan inovasi baru, bahkan saya tidak dapat melihat efeknya. ”

Mengaktifkan cacing Gu yang tidak dikenal adalah hal yang berbahaya.

Misalnya, Spring Autumn Cicada adalah contoh terbaik.

Tapi Surga Pencuri muda baru saja mulai berkultivasi dan tidak tahu apa-apa, dan Sha Xiao, yang tahu tentang risikonya, tidak peduli dengan keselamatan Surga Pencuri muda.

Surga Pencuri yang begitu muda dengan bodohnya mengaktifkan cacing Gu ini.

Saat diaktifkan, cacing Gu ini berubah kembali menjadi bayi hantu biru yang mengeluarkan jeritan aneh sebelum keluar dari lubang Pencuri Surga muda dan menyerbu ke depan.

Ada bangku batu di depan, hantu bayi biru langsung memasuki bangku batu, lalu keluar dan kembali ke lubang muda Pencuri Surga, kembali ke penampilan Gu-nya.

Bangku batu itu tampak tidak rusak.

Surga Pencuri Muda berjalan ke bangku dan menyentuhnya.

Bam.

Suara renyah bergema saat bangku batu berubah menjadi potongan-potongan batu yang hancur.

Surga Pencuri Muda terkejut dan mundur selangkah, melihat tumpukan kerikil di tanah dengan kaget.

“Hebat!” Sha Xiao memuji: “Cacing Gu ini memiliki kekuatan yang sangat tinggi, ini adalah cacing Gu peringkat dua tingkat atas. Lebih penting lagi, sepertinya bisa melewati pertahanan, lihat, bangku batu itu hancur dari dalam. ”

Sha Xiao cerdik, melihat banyak kedalamannya dalam sekejap.

Surga Pencuri Muda masih agak kacau.

Sha Xiao melihat aperture-nya dan memuji lagi: “Luar biasa! Lihatlah aperture Anda, hanya sepuluh persen esensi purba yang telah dikeluarkan untuk mengaktifkan cacing Gu ini, ini tidak terbayangkan. Ini adalah cacing Gu peringkat dua dengan begitu banyak kekuatan, tetapi hanya membutuhkan sepuluh persen esensi purba tembaga hijau peringkat satu!

“Tapi aku hanya memiliki bakat kelas D, esensi purbaku hanya sedikit lebih dari dua puluh persen. Saya hanya bisa menggunakan cacing Gu ini dua kali.” Surga Pencuri Muda khawatir.

“Kamu tidak tahu apa-apa. Dengan cacing Gu ini, posisi pertama dalam kompetisi adalah milikmu. Tidak ada yang bisa bertahan melawan cacing Gu Anda. ” Sha Xiao mendengus.

“Aku bisa menjadi yang pertama dengan Gu ini?” Mata Pencuri Muda Surga terbuka lebar.

“Begitu worm Gu ini diaktifkan, ia akan berubah menjadi baby phantom biru yang akan menyerang musuh. Anda telah melihat kecepatannya, sangat cepat. Peringkat tiga Gu Masters mungkin bisa mengelak dengan susah payah, tetapi melawan peringkat satu dan peringkat dua Gu Masters, selama Anda menutup jarak, mereka akan diserang dan tidak punya waktu untuk menghindar. ”

Sha Xiao terus menganalisis, bergumam: “Hmm… melihat kekuatan dan efek cacing Gu ini, kita mungkin juga menyebutnya serangan menyelinap Gu.”

Cacing Gu diciptakan oleh Surga Pencuri muda, tetapi Sha Xiao memberanikan diri untuk menamai cacing Gu ini.

“Serangan menyelinap Gu …” Surga Pencuri Muda segera mengerutkan kening dan ingin keberatan.

“Ya, sebut saja serangan menyelinap Gu, indra penamaanku masih bagus. Ha ha ha!” Sha Xiao berkata dengan dominan, Surga Pencuri muda hanya bisa memutar matanya tanpa daya.
