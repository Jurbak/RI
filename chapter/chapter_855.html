Suara mendesing!

Cahaya keemasan sangat cepat, mencapai Fang Yuan dalam sekejap mata, menargetkan api pemurnian di tangan Fang Yuan.

Jika cahaya keemasan mengenai api, bahkan dengan kemampuan Fang Yuan, penyempurnaan akan gagal, dia harus menanggung serangan balik dan memulai dari awal.

Ada keributan di luar panggung, tetapi di dalam panggung, pemandangannya sunyi, terisolasi dari semua suara di luar.

‘Menyerang titik vital musuh, lalu menggunakan momen relatif aman ini untuk menggunakan gerakan pembunuh jalur penyempurnaan lainnya?’ Fang Yuan melirik Feng Jin Huang, dengan dingin tersenyum ke dalam, ‘Sepertinya kamu menggunakan semua kekuatanmu. Layak menjadi Feng Jin Huang, bahkan di masa muda Anda, Anda sangat menyadari kekuatan dan kelemahan Anda sendiri. Saya harus mengatakan ini adalah pilihan yang masuk akal.’

Fang Yuan memujinya dalam pikirannya, tetapi dia terus tidak tergerak, masih memurnikan Gu dengan cara yang sama seperti sebelumnya.

Feng Jin Huang menggunakan kesempatan berharga untuk bergerak menyerang api pemurnian Fang Yuan, dan juga mengambil kesempatan ini untuk menggunakan langkah pembunuh jalur penyempurnaan kedua.

Dengan bantuan dua gerakan pembunuh, kecepatan pemurniannya meningkat tajam, dan secara bertahap Fang Yuan terlempar ke belakang.

Tanpa tekanan dari Fang Yuan, Feng Jin Huang segera merasa sedikit rileks, lapisan ketegangan di dalam dirinya mengendur, dan kecepatan penyempurnaannya semakin meningkat sedikit.

Dalam duel ini, Fang Yuan dan Feng Jin Huang masing-masing memiliki lima peluang untuk menyerang.

Karena itu hanya duel, bukan pertarungan hidup dan mati, kekuatan yang bisa mereka gunakan setiap kali menyerang dibatasi pada level tertentu.

Pertama kali, mereka hanya diizinkan menggunakan Gu peringkat satu. Kedua kalinya, mereka bisa menggunakan peringkat dua Gu. Dan seterusnya, sampai kelima kalinya, di mana mereka bisa menggunakan peringkat lima Gu.

Itu sama dengan pertahanan, mereka hanya bisa menggunakan cacing Gu dengan peringkat yang sesuai.

Ini adalah batasan lain untuk Fang Yuan.

Bagaimanapun, Fang Yuan adalah zombie abadi, dengan gerakan pembunuh abadi. Jika dia diserang oleh tangan raksasa, akan sulit bagi Feng Jin Huang untuk menolaknya. Tentu saja, Feng Jin Huang memiliki orang tua Gu Immortal, jadi dia secara alami akan memiliki esensi abadi di tubuhnya bersama dengan Immortal Gu dan kehendak atau emosi dari Gu Immortals. Tidak ada kemungkinan besar Fang Yuan benar-benar membunuhnya.

“Kali ini, dia menggunakan dua gerakan pembunuh jalur penyempurnaan sekaligus untuk mendapatkan petunjuk!” Di luar panggung, Sun Yao berteriak dengan penuh semangat.

“Ya, sangat disayangkan dia menggunakan salah satu dari lima peluang untuk menyerang. Tapi ini juga taktik klasik, kita masih harus melihat hasil penyempurnaan untuk menentukan hasilnya.” Qin Juan merenung saat dia menatap panggung.

Waktu terus berlalu. Fang Yuan jatuh semakin jauh di belakang Feng Jin Huang, tetapi bagian yang aneh adalah dia tidak sedikit pun terburu-buru, tampaknya tidak memiliki niat untuk menyalip Feng Jin Huang.

Fang Yuan masih menggunakan teknik penyempurnaan Gu, menyempurnakan arang lubang tunggal Gu langkah demi langkah.

Sementara Feng Jin Huang sudah mulai memperbaiki tungku lubang ganda peringkat dua Gu.

Segera, pujian dan desahan terdengar di luar panggung.

Feng Jin Huang telah menggunakan gerakan pembunuh jalur penyempurnaan ketiga, meningkatkan kecepatannya sekali lagi.

Satu jam kemudian, Feng Jin Huang menyerang Fang Yuan dan mengambil kesempatan ketika Fang Yuan bertahan, dia menggunakan gerakan pembunuh jalur penyempurnaan keempat.

Empat gerakan pembunuh digunakan bersama-sama, ada enam gumpalan api yang mengambang di depan Feng Jin Huang, setiap gumpalan disertai dengan tangan angan-angan kecil yang terus-menerus mengambil bahan dan melemparkannya ke dalam api.

Dengan bantuan angan-angan kecil, jumlah yang tepat dari bahan penyempurnaan Gu yang dilemparkan ke dalam api sangat tepat.

Dari bantuan empat gerakan pembunuh, kecepatan penyempurnaan Feng Jin Huang telah lama melampaui kecepatan Fang Yuan, dan segera dia mulai dengan tiga lubang api di gubuk Gu.

Sementara Fang Yuan baru saja mulai memperbaiki tungku lubang ganda Gu.

Melihat Feng Jin Huang maju dengan penuh kemenangan, melemparkan Fang Yuan jauh ke belakang, banyak yang ragu: Apa yang sedang dimainkan Fang Yuan?

Sun Yao menyeringai: “Kemenangan kakak perempuan pertama dijamin!”

Qin Juan juga seorang jenius jalur penyempurnaan yang sangat diperhatikan oleh Spirit Affinity House, hanya saja dia belum dewasa. Dia juga mendukung kata-kata Sun Yao: “Jika situasinya terus seperti ini, kakak perempuan pertama memiliki peluang besar untuk menang.”

“Dengan kemampuanku, batas maksimumku adalah mempertahankan lima gerakan pembunuh jalur penyempurnaan. Tapi ini adalah format duel bela diri, saya harus meninggalkan energi untuk menghadapi serangan Fang Yuan. Dari lima peluang untuk menyerang, saya sudah menggunakan dua, sementara Fang Yuan hanya bertahan. Mungkinkah kondisi saya terlalu memikat sehingga dia kehilangan?

Feng Jin Huang tidak bisa memahami perilaku Fang Yuan dan merasa ragu sejenak.

Lima belas menit berlalu, Feng Jin Huang telah membuat kemajuan yang baik dalam memperbaiki tiga lubang api tetapi Gu, dia melewati tanda setengah jalan.

Saat ini, dominasi Feng Jin Huang sangat jelas, memimpin Fang Yuan dengan jarak yang sangat jauh!

‘Mm … sekarang waktunya untuk menyerang.’ Tatapan Fang Yuan jelas, dan ketika dia merasakan bahwa waktunya sudah matang, dia mengirim cacing Gu dari lubang abadinya.

“Aku menyerang.” Dia tiba-tiba berbicara dengan lembut.

“Eh.” Perhatian Feng Jin Huang langsung tertarik, matanya yang cerah dan indah menatap Fang Yuan, sambil melanjutkan penyempurnaan Gu-nya.

‘Akhirnya menyerang? Tapi apa alasan untuk memperingatkan saya sebelumnya?’ Feng Jin Huang sangat berhati-hati.

Fang Yuan terkekeh: “Feng Jin Huang, hati-hati. Gu ini adalah Gu api hantu peringkat satu.”

Feng Jin Huang menyipitkan matanya, tatapan waspada melintas di matanya saat dia mulai berpikir cepat: ‘Sudah pasti Gu peringkat satu, ini adalah serangan pertama Fang Yuan, jadi dia hanya bisa menggunakan metode peringkat satu. Tapi itu mungkin belum tentu Gu api hantu, mungkin dia menunjukkan bahwa Gu menyesatkan saya? Bagaimanapun, penampilan luar cacing Gu bisa disamarkan. Saya harus sangat berhati-hati!’

Sementara dia berpikir, Fang Yuan mengaktifkan cacing Gu di tangannya.

Saat Gu ini diaktifkan, itu berubah menjadi gumpalan kecil api hantu yang menakutkan, yang kemudian perlahan bergerak menuju Feng Jin Huang.

‘Ini benar-benar hantu api Gu?’ Feng Jin Huang sedikit tercengang.

Tepat pada saat ini, api hantu Gu tiba-tiba melesat dan menembak ke arah Feng Jin Huang!

‘Datang! Hanya hantu api peringkat satu Gu …’ Mata Feng Jin Huang bersinar tajam dengan percaya diri. Dia sudah cukup siap dan dapat dengan mudah memblokir Gu api hantu peringkat satu.

Ketika api hantu Gu menyerang, dia bahkan punya waktu untuk memikirkan kemungkinan Fang Yuan melakukan serangan terus menerus!

Namun, api hantu tiba-tiba berhenti, berhenti di dekat Feng Jin Huang.

Setelah itu, api hantu perlahan mengitari Feng Jin Huang, bergoyang di kedua sisinya.

‘Apa?’ Feng Jin Huang terkejut secara internal: ‘Fang Yuan, apa artinya ini?!’

Saat berikutnya, ekspresinya tenggelam saat kesadaran muncul di benaknya.

Dalam duel ini, Gu Master hanya bisa bertahan dalam jarak satu meter. Api hantu Gu tidak melanggar jarak satu meter di sekitar Feng Jin Huang, membuatnya tidak bisa bertahan. Api hantu yang melayang-layang hanya bisa dianggap sebagai bagian dari proses serangan Fang Yuan.

‘Dia menggunakan api hantu peringkat satu untuk menekanku, untuk mengalihkan perhatianku!’ Feng Jin Huang cerdik dan segera bisa memahami maksud di balik langkah ini.

“Sangat jahat!” Banyak orang, di luar panggung, berseru ketika mereka menyadari rencananya.

Sun Yao dipenuhi dengan kemarahan yang benar: “Iblis Fang Yuan itu menggunakan celah dalam aturan. Dengan api hantu melayang di depanmu, siapa yang bisa berkonsentrasi untuk menyempurnakan Gu? Apakah tidak ada yang peduli tentang ini? Dia terlalu tidak tahu malu.”

Qin Juan menggelengkan kepalanya, berkata dengan sungguh-sungguh: “Aturan duel ditetapkan oleh kedua belah pihak. Serangan yang tertunda ini juga merupakan taktik pertempuran klasik dalam format duel bela diri, itu tidak bisa dianggap tidak tahu malu.”

‘Brengsek!’ Keringat dingin mulai merembes keluar dari dahi Feng Jin Huang, ‘Api hantu bisa melukai jiwa. Saya telah menghabiskan banyak energi mental saya, jika saya benar-benar terpengaruh olehnya, saya pasti akan tergelincir. Tetapi jika saya hanya membela secara membabi buta, pikiran saya akan terikat dalam mengamati pergerakan api hantu, dan kecepatan penyempurnaan Gu saya akan berkurang.’

Masih ada jalan, Feng Jin Huang bisa memulai serangan dan menghancurkan api hantu ini.

Tapi ini tidak berharga.

Karena ini akan menjadi serangan ketiganya, yang berarti metode peringkat tiga. Menggunakan metode peringkat tiga untuk membatalkan Gu api hantu peringkat satu Fang Yuan, tidak peduli apa, itu akan menjadi kerugiannya!

‘Mungkin ini niatnya, dia ingin memancingku menggunakan kesempatan berharga untuk menyerang!’ Feng Jin Huang menggertakkan giginya saat dia melihat Fang Yuan.

Mengontrol api hantu juga menghabiskan energi mental seseorang. Tapi Fang Yuan hanya menggunakan teknik pemurnian Gu dan bukan gerakan pembunuh jalur penyempurnaan, bisa dikatakan energi mentalnya dalam kapasitas penuh, itu lebih dari cukup untuk mengendalikan api hantu.

“Oh? Tidak menghancurkan api hantu saya? Hehehe, kamu benar-benar sabar. ” Fang Yuan tiba-tiba tertawa, “Lalu, bagaimana dengan api hantu lainnya?”

Di bawah tatapan semua orang, Fang Yuan mengeluarkan Gu api hantu kedua.

“Feng Jin Huang, perhatikan baik-baik, ini adalah peringkat dua api hantu Gu.” Fang Yuan dengan tenang berbicara, tidak seperti serangan diam-diam Feng Jin Huang, dia secara terbuka mengumumkan metodenya.

Gaya serangan Fang Yuan benar-benar kebalikan dari Feng Jin Huang.

Dua serangan Feng Jin Huang cepat, akurat dan tanpa henti, menyerang titik vital musuh dan memaksa Fang Yuan untuk bertahan, sementara dia mengambil kesempatan untuk menggunakan gerakan pembunuh jalur penyempurnaan.

Namun, metode Fang Yuan dapat dijelaskan dengan satu kata — lambat.

Api hantu kedua melayang perlahan ke arah Feng Jin Huang, berhenti satu meter di depannya.

Api hantu ini dibuat dari peringkat dua api hantu Gu, ukurannya dua kali lipat dari api hantu pertama.

Dua api hantu berputar di sekitar Feng Jin Huang, tekanan pada Feng Jin Huang dengan cepat berlipat ganda!
