Harta karun surga kuning.

Benjolan besar api transparan diam-diam menyala sambil mengeluarkan cahaya harta karun yang besar.

Api segera menarik perhatian banyak Dewa Gu saat muncul.

Banyak keinginan dan beberapa indera ilahi tertarik ke tempat kejadian.

Gu Immortal Zhang Ji dari Kuil Surga Hitam tersenyum, dia adalah pemilik api transparan ini.

“Saya tidak pernah berharap ekspedisi ini memberi saya sekumpulan materi abadi peringkat tujuh yang langka. Keuntungannya seharusnya cukup untuk mengisi kembali cadangan esensi abadi saya. ”

Sambil berpikir, dia menjawab pertanyaan semua orang tentang harganya.

Harganya hanya sedikit lebih tinggi dari biasanya, sehingga banyak yang berpikir untuk membeli.

“Aku ingin semua ini.” Satu indera ilahi langsung menghubungi Zhang Ji, menawarkan harga tinggi.

Jantung Zhang Ji berdebar kencang, agak tidak percaya: “Kamu menginginkan semua ini? Harga…”

“Buat kesepakatan.” Sisi lain sangat mudah, langsung memberikan sejumlah besar batu esensi abadi.

“Oke!” Melihat sejumlah besar batu esensi abadi, mata Zhang Ji langsung cerah, langsung setuju.

Transaksi selesai dengan cepat, Dewa Gu yang mengamati api agak lengah.

“Tidak perlu khawatir, saya masih punya stok!” Perasaan ilahi Zhang Ji terus berfluktuasi, dia dengan cepat menyatakan dan mengeluarkan gumpalan api transparan lainnya.

Hanya saja jumlah api transparan ini jauh lebih kecil dari bongkahan yang dijual sebelumnya.

Transaksi pertama telah merangsang keinginan Dewa Gu lainnya untuk membeli, lagipula, api transparan ini memang langka di pasar. Biasanya, mereka hanya dapat mengakumulasikannya selama pembelian terus menerus, dan setiap pembelian biasanya hanya seukuran kepalan tangan atau seukuran baskom api. Mereka berbeda dari sekarang di mana api transparan seukuran kereta! Mereka tidak bisa kehilangan kesempatan langka ini.

“Tapi kali ini, aku akan menaikkan harganya.” Senyum Zhang Ji menjadi lebih cerah. Dia tidak memiliki banyak pengetahuan tentang api transparan. Ini juga pertama kalinya dia menjualnya dan dia tidak memiliki pemahaman yang tepat tentang situasi pasar, sehingga antusiasme Dewa Gu lainnya melampaui harapannya.

Dia terus menaikkan harga, tetapi beberapa pembeli menyerah, sehingga harga awal terus menjadi lebih tinggi.

“Sialan, sepertinya aku melakukan kesalahan dalam menjual sebelumnya!” Zhang Ji melihat pemandangan ini dengan sedikit penyesalan, sambil juga mengagumi ketegasan pembeli itu.

Zhang Ji tidak bodoh, dia telah membagi persediaan api transparannya menjadi tiga bagian. Bagian pertama adalah untuk menguji air, sedangkan yang dia jual sekarang adalah bagian kedua.

Ketika transaksi terus meningkat, dia mendapatkan pemahaman yang lebih jelas tentang situasi pasar.

“Api kehidupan mengambang!” Fang Yuan melihat gumpalan api transparan ini di lubang abadinya yang berdaulat dengan ekspresi senang.

Dialah yang memimpin dan membeli barang-barang Zhang Ji.

Api kehidupan yang mengambang ini transparan dan menyala tanpa suara tanpa panas. Tetapi ketika membakar makhluk hidup, semua jenis adegan masa lalu makhluk hidup ini akan berubah menjadi gambar yang terus-menerus diputar ulang dalam api kehidupan yang mengambang. Pada saat itu, api kehidupan yang mengambang akan memancarkan cahaya berwarna-warni, dan tidak lagi transparan.

Api kehidupan terapung perlu dilestarikan dengan metode khusus.

Zhang Ji secara kebetulan mendapatkan sejumlah besar api kehidupan mengambang, dia tidak memiliki metode seperti itu.

Setelah Fang Yuan mendapatkannya, dia segera menggunakan metode untuk melestarikan api kehidupan yang mengambang. Metode ini bukan dari warisan sejati Spectral Soul, tetapi diperoleh dari warisan sejati Lang Ya yang telah ditukar dengan warisan sejati Spectral Soul.

“Saya pikir saya perlu melakukan perjalanan jauh dan menemukan masalah dengan orang lain untuk merebut api kehidupan yang mengambang ini. Siapa sangka ada orang yang menjualnya di harta surga kuning dalam jumlah yang begitu besar sekarang. Keberuntungan ini benar-benar luar biasa!”

Fang Yuan tersenyum.

Pasti ada banyak Dewa Gu yang ingin membeli api kehidupan mengambang.

Tapi wasiat Gu Immortal biasanya tidak dapat membuat keputusan, mereka hanya bisa mengikuti perintah yang ditinggalkan oleh Gu Immortal.

Dan untuk bisa mengeluarkan begitu banyak batu esensi abadi sekaligus juga bukan sesuatu yang bisa dilakukan Dewa Gu biasa.

Bahkan jika mereka memiliki modal, mereka yang mengamati Dewa Abadi ingin menawar karena jumlah yang mereka butuhkan biasanya tidak sebesar itu.

Karena banyak alasan, Fang Yuan dapat melakukan langkah pertama dan mendapatkannya dengan harga murah!

Adapun risikonya?

Zhang Ji ini adalah Gu Immortal dari Kuil Surga Hitam, seseorang dari Pengadilan Surgawi, tetapi dia tidak tahu pembelinya adalah Fang Yuan, Fang Yuan juga tidak tahu identitasnya.

Transaksi di Treasure Yellow Heaven sangat aman. Jika Pengadilan Surgawi dapat menggunakan ini untuk merencanakan melawan Fang Yuan, untuk memiliki pengaruh yang begitu dalam bahkan di surga kuning harta karun, maka lebih baik bagi Fang Yuan untuk menyerah saja.

Secara keseluruhan, kemungkinan ini agak rendah.

Minum anggur bilah, Gu tepi, jarum penghalang ilahi depan-belakang, dan api kehidupan mengambang, keempat bahan abadi utama ini semuanya dikumpulkan. Fang Yuan masih harus membeli lebih banyak tepi Gu, yang dapat sepenuhnya dipenuhi oleh surga kuning harta karun, jika tidak, ia masih memiliki saluran yang tersisa di klan Xiao Gurun Barat.

Karena tidak ada artinya tinggal di luar, Fang Yuan bergegas kembali ke tanah yang diberkati Lang Ya.

Melewati formasi abadi, Fang Yuan kembali ke tanah yang diberkati Lang Ya. Prosesnya lancar tanpa bahaya.

Kembalinya Fang Yuan jauh lebih awal dari yang diharapkan, ini membuat roh tanah Lang Ya sangat bahagia.

“Karena kamu telah kembali, aku akan bersiap untuk penyempurnaan.” Roh tanah Lang Ya sangat aktif, tekanan dari Pengadilan Surgawi benar-benar terlalu besar.

“Saya masih perlu berterima kasih kepada tetua tertinggi pertama dan Dewa Gu dari tiga ras lainnya karena telah meminjamkan saya begitu banyak batu esensi abadi.” Fang Yuan berterima kasih.

Tanpa bantuan mereka, bagaimana mungkin Fang Yuan mengeluarkan jumlah yang begitu besar?

“Jangan berdiri di atas upacara. Kita semua berada di pihak yang sama, kamu semakin kuat berarti tanah yang diberkati Lang Ya akan lebih aman. Jika tanah yang diberkati Lang Ya dalam bahaya, mereka juga tidak akan bisa melarikan diri karena perjanjian aliansi! ” Balasan roh tanah Lang Ya langsung dan jujur.

Memang, situasi saat ini membuatnya sedemikian rupa sehingga Sekte Lang Ya serta manusia salju, manusia batu, dan manusia tinta berada di kapal yang sama. Jika ada yang menghadapi bahaya, sisanya tidak akan bisa melarikan diri.

Fang Yuan menyerahkan empat bahan abadi ke Sekte Lang Ya untuk diproses.

Bahan abadi ini membutuhkan setidaknya lima belas hari pemrosesan, langkah-langkahnya sangat membosankan, berulang dan rumit. Ini adalah kasus ketika semua pria berbulu Gu Dewa dari Sekte Lang Ya mengerahkan upaya penuh mereka.

Keunggulan dari segudang resep Immortal Gu bisa dilihat dari ini!

Dengan pencapaian grandmaster kuasi-tertinggi jalur penyempurnaannya, resep Immortal Gu yang tak terhitung jumlahnya dari Fang Yuan begitu luar biasa sehingga bisa dikatakan telah mencapai batas dunia tanpa ruang tersisa untuk perbaikan.

Prosedur penting dari penyempurnaan Gu dipersingkat hingga batasnya, totalnya ada tiga. Sementara prosedur lain memiliki lebih dari sepuluh ribu langkah, kebanyakan dari mereka adalah pemrosesan bahan abadi.

Fang Yuan memproses bahan abadi ke kondisi terbaik mereka, menghabiskan sebagian besar usahanya dalam hal ini. Bahkan jika pengeluaran bahan abadi meningkat karena ini, dia tidak ragu-ragu.

Karena prosedur ini adalah prosedur yang bisa dikendalikan oleh Dewa Abadi. Sementara langkah-langkah penting dari penyempurnaan Gu hanya bisa mengandalkan tabrakan dan jalinan tanda dao, Dewa Gu tidak bisa mengendalikan perubahan ini.

Melalui metode ini dan dengan bantuan kemampuan penyempurnaan Gu Sekte Lang Ya, tingkat keberhasilan resep Immortal Gu yang tak terhitung jumlahnya milik Fang Yuan berada pada lima puluh persen yang sangat tinggi!

Itu berarti Fang Yuan memiliki kesempatan yang sama untuk berhasil atau gagal dalam memperbaikinya.

Tingkat keberhasilan lima puluh persen benar-benar terlalu menakutkan.

Mayoritas mutlak Dewa Gu akan senang jika mereka memiliki tingkat keberhasilan bahkan satu persen saat menyempurnakan Gu.

Namun, ini juga terutama karena Fang Yuan adalah grandmaster kuasi-tertinggi jalur penyempurnaan!

Berapa banyak grandmaster kuasi-tertinggi jalur penyempurnaan yang ada di dunia? Sejak zaman kuno hingga sekarang, seorang ahli di tingkat Fang Yuan dapat dihitung dengan jari. Mereka yang melampaui levelnya hanya berjumlah tiga orang sepanjang sejarah.

Selain itu, ada sangat sedikit Dewa Gu yang mengolah banyak jalur, dan berapa banyak dari mereka yang akan mengolah jalur penyempurnaan?

Dari sudut lain, sebagian besar Dewa Gu sebenarnya adalah amatir dalam menyempurnakan Gu Abadi. Akan tidak masuk akal jika mereka bisa meningkatkan tingkat keberhasilan penyempurnaan Gu!

Faktanya, bahkan Fang Yuan, yang telah menyimpulkannya, sedikit tidak percaya dengan tingkat keberhasilan lima puluh persen.

Tapi kemudian dia memikirkan Leluhur Rambut Panjang.

Leluhur Rambut Panjang telah menyempurnakan banyak Gu Abadi dalam hidupnya, termasuk peringkat tujuh dan peringkat delapan Gu Abadi. Dia adalah grandmaster tertinggi jalur penyempurnaan, jika dia tidak memiliki tingkat keberhasilan seperti itu, akankah Giant Sun Immortal Venerable dan Thiieving Heaven Demon Venerable menemukannya untuk meminta bantuan?

“Selanjutnya, saya menggunakan seluruh kekuatan Sekte Lang Ya untuk memperbaiki Gu ini. Di zaman sekarang, ketika membandingkan pencapaian jalur penyempurnaan, Sekte Lang Ya tidak diragukan lagi menjadi kekuatan nomor satu.”

“Selain ini, alasan utama ketiga adalah keberuntunganku yang kuat.”

Pertama adalah tingkat pencapaian grandmaster kuasi-tertinggi jalur pemurniannya, kedua adalah bantuan habis-habisan kekuatan pemurnian nomor satu, dan ketiga adalah keberuntungan kuat yang diperoleh dari warisan sejati Giant Sun.

Tiga alasan besar digabungkan untuk membentuk tingkat keberhasilan lima puluh persen dari segudang penyempurnaan Immortal Gu!

“Saya akhirnya bebas dari kutukan pemurnian Gu Abadi.” Fang Yuan sangat tersentuh sampai memiliki keinginan untuk menangis.

Dia tahu alasan terpenting adalah dia menjadi grandmaster kuasi-tertinggi jalur penyempurnaan!

Dua alasan lainnya sangat membantu, tetapi mereka kurang lebih hadir di semua penyempurnaan Gu sebelumnya. Tingkat keberhasilannya yang lima puluh persen sebagian besar masih dibawa oleh pencapaian grandmaster kuasi-tertinggi jalur penyempurnaan.

“Dan inilah saat saya hanya menyimpulkan dan memodifikasi resep Immortal Gu. Di masa depan, ketika saya menebus kurangnya keterampilan dasar saya dalam penyempurnaan Gu, saya secara pribadi dapat bertanggung jawab atas proses penyempurnaan, tingkat keberhasilan akan meningkat lebih banyak lagi!

“Sayang sekali aku tidak punya cukup waktu…”
